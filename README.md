# 📊 Global GDP Analysis (1960-2016)

A comprehensive data analysis project exploring worldwide GDP trends, economic growth patterns, and country-level comparisons using Python.

## 🎯 Project Overview

This project analyzes historical GDP data from 1960 to 2016 for countries and regions worldwide. The analysis focuses on identifying economic trends, growth patterns, and regional comparisons using data visualization and statistical analysis.

## 📁 Dataset

- **Source**: World Bank GDP Data
- **Time Period**: 1960-2016
- **Coverage**: 256 countries and regions
- **Total Records**: 11,507 data points
- **Format**: CSV with columns for Country Name, Country Code, Year, and GDP Value (current USD)

### Data Cleaning

The dataset initially included aggregate regions (e.g., "Arab World", "Euro area") which were filtered out to focus on individual country analysis. The final dataset contains **9,201 rows** of country-specific data.

## 🛠️ Technologies Used

- **Python 3.11
- **pandas** - Data manipulation and analysis
- **numpy** - Numerical computations
- **matplotlib** - Data visualization

## 📈 Key Features

- Data cleaning and preprocessing to remove regional aggregates
- Exploratory data analysis (EDA) of GDP trends
- Time series visualization of country-specific GDP growth
- Identification of economic patterns and recessions
- Comparative analysis across countries and decades
